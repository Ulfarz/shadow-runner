# Shadow Runner Project Overview

## Purpose
Shadow Runner is a mobile application, likely a game, developed using web technologies and packaged for Android using Capacitor.

## Tech Stack
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, PostCSS
- **Mobile Packaging**: Capacitor
- **Backend/Database**: Supabase
- **Other**: Firebase (detected `.firebaserc` and `firebase.json`)

## Codebase Structure
- `src/`: Frontend source code (React/TS).
- `android/`: Android native project (Capacitor).
- `public/`: Static assets.
- `supabase_schema.sql`: Database schema for Supabase.
